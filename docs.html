Log style samples for your customs worksheet in a few taps. Set up your API key first — everything else takes seconds.

1

Add your API key Do this first

The Anthropic API key unlocks Photo Scan — the feature that reads your tags from photos and fills in the sample details for you.

How to add it

  1. Tap the ⚙ gear icon in the top-right corner.
  2. Paste your key into the Anthropic API key box.
  3. That's it — it saves automatically and checks itself.

Did it work? Check the pill

A small pill sits at the top of the app and tells you the key's status:

Connected No key

Green = ready to scan. If it's red, tap the pill to try again. The label tells you why:

No key  Nothing entered yet — add your key.
Invalid key  The key was rejected. Re-copy and paste it carefully.
Offline  The app couldn't reach the internet/API at all.
Rate limited / Check failed  Wait a few seconds, then tap the pill to retry.
⚠️
Stuck on "Offline"? Check your connection. If you opened the app by double-clicking the file, open it through a web address instead — or add it to your home screen — so it can reach the API.
🔒
Your key is stored only on this device. It's never shared with anyone but Anthropic when you scan.
2

Add a sample

Tap + Add sample. There are two ways to fill it in:

📷 Scan with photos (fastest)

  1. Take or upload up to four photos: the price tag, the content/care label, the garment, and the country of origin.
  2. Tap Scan & auto-fill.
  3. Claude reads the labels and fills the form. Review the details, fix anything that looks off, then tap Save.

✍️ Enter it manually

  • Switch to manual entry and type the details yourself.
  • You can still attach a photo if you'd like one on file.
3

Review & edit

Saved samples appear in the list. The pills at the top keep a running total of your sample count, total quantity, and value by currency.

Tap any sample to edit it, or remove it if it was added by mistake.

4

Export & send

  • Save creates an Excel file (.xlsx) with all your sample details plus thumbnail photos.
  • You can also export a ZIP of the full-resolution photos, named by style.
  • On a phone, Share sends the file straight to email, AirDrop, or messaging. Set a default address in Settings so it's pre-filled.
5

Continue a list later

Use Import existing .xlsx and pick a file you exported earlier to pick up right where you left off.

6

Settings (⚙)

  • Send-to email — pre-fills the share recipient.
  • Anthropic API key — for Photo Scan (see step 1).
  • View documentation — opens this help page.
  • Clear all samples — wipes everything and starts fresh.
Everything is saved on your device only. Photos are sent to Anthropic just to read the labels during a scan — nothing else leaves your phone.
